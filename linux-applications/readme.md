# Filesystem

## Disk usage

* ncdu - ncurses filesystem manager. (CLI)
* baobab - list/pie chart interface (GUI)
* k4dirstat - node-like layout (KDE, GUI)
* gdmap - node-like layout (GUI)
* jdiskreport - list/piechart interface (JAVA, GUI)
