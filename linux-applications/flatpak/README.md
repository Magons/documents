# Example build
Sourced from: http://docs.flatpak.org/en/latest/building-simple-apps.html

clone source code:
```
git clone <URL>
```

Build format:
```
flatpak build-init DIRECTORY APPNAME SDK RUNTIME [BRANCH]
```
